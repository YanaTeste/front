-- SUMMARY --

Multilingual logo support.

For a full description of the module, visit the project page:
  http://drupal.org/project/i18nlogo

To submit bug reports and feature suggestions, or to track changes:
  http://drupal.org/project/issues/i18nlogo


-- REQUIREMENTS --

Locale.


-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.


-- CONFIGURATION --

* Configure user permissions in Administer >> User management >> Permissions 

  - administer languages

    Users who can administer the languages will be able to upload a logo for 
    each language.

* Add one or more language in Administer >> Site configuration >> Languages


-- CUSTOMIZATION --

There is nothing to customize in this module.
The logo is being displayed by your page.tpl.php as usual.

-- TROUBLESHOOTING --

* If the logo does not show, check the folloeing

  - Are you looking at the appropriate language?

  - Make sure your sites/default/files is readable and writable.
